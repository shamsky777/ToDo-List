# ToDo List Website

## Description

This is a simple ToDo list web application where users can manage their tasks efficiently. It allows users to create and delete tasks. The application provides a clean and intuitive interface to help users stay organized and productive.

## Features

- **Create Tasks**: Add new tasks with a title and optional description.
- **Delete Tasks**: Remove unwanted tasks from the list.
- **Future Features**: Planned features include "Mark as Completed" and "Edit" buttons for enhanced task management.

## Installation

This ToDo list application is hosted online and does not require any installation. You can access the website by clicking on the link below:

[Open ToDo List Website](https://todokkk.netlify.app/)

## Usage

1. Open the ToDo list website by clicking on the link provided above.
2. Use the input field at the top to add new tasks. Press Enter or click the "Add" button to create a task.
3. To delete a task, click on the trash icon next to the task.
4. Stay tuned for future updates, which will introduce additional features such as "Mark as Completed" and "Edit" buttons for better task management.
